Como você quer guardar seus registros?

Crie uma senha para proteger seus dados. Guarde-a bem: sem ela seus dados não podem ser recuperados.
Como usar com o GitHub
  1. Crie uma conta gratuita em github.com
  2. Crie um repositório novo (pode ser público ou privado)
  3. Faça o upload do index.html para o repositório
  4. Gere um Personal Access Token com permissão repo
  5. Ative o GitHub Pages em Settings → Pages do repositório
  6. Acesse o app pela URL gerada e informe suas credenciais
Como usar localmente

Os dados ficam salvos neste navegador, neste dispositivo. Sem necessidade de conta em nenhum serviço.
